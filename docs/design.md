# Anime Crossover Fighter - Game Design Document

## Overview

A traditional 2D fighting game built in Ikemen Go featuring 16 characters from 8 anime universes. Players battle through dimensions, unlocking fighters and eventually creating custom characters by mixing abilities.

## Roster (16 Characters)

| Anime | Character 1 | Character 2 |
|-------|-------------|-------------|
| Naruto | Naruto | Sasuke |
| Dragon Ball Z | Goku | Vegeta |
| Demon Slayer | Tanjiro | Upper Moon Demon |
| My Hero Academia | Deku | Bakugo |
| One Piece | Luffy | Zoro |

## Game Mode

- Dimension-hopping journey through all 8 anime universes
- Start with Naruto unlocked
- Defeat characters to unlock them
- Progressive difficulty scaling (Naruto universe = easiest, One Punch Man = hardest)

**Progression Order:**
1. Naruto Universe → Fight Sasuke → Unlock Sasuke
2. Dragon Ball Z → Fight Goku → Fight Vegeta → Unlock both
3. Demon Slayer → Fight Tanjiro → Fight Demon → Unlock both
4. My Hero Academia → Fight Deku → Fight Bakugo → Unlock both
5. One Piece → Fight Luffy → Fight Zoro → Unlock both
6. Attack on Titan → Fight Eren → Fight Levi → Unlock both
7. Jujutsu Kaisen → Fight Gojo → Fight Sukuna → Unlock both
8. One Punch Man → Fight Genos → Fight Saitama (final boss) → Unlock both

## Art Style & Assets

- **Visual Style**: Hand-drawn anime-style sprites
- **Asset Source**: MUGEN community sprite sheets
- **Stages**: 8 backgrounds (one per anime universe), sourced from community

### Stages
1. Hidden Leaf Village (Naruto)
2. World Tournament Arena (Dragon Ball Z)
3. Demon Slayer Forest/Mountain
4. UA High School (My Hero Academia)
5. Thousand Sunny / Marine Ford (One Piece)
6. Wall Maria / Titan Territory (Attack on Titan)
7. Shibuya / Jujutsu High (Jujutsu Kaisen)
8. City Z (One Punch Man)
